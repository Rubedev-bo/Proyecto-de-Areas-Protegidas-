# Sistema Web de Gestión de Areas Protegidas

El presente proyecto busca mostrar el manejo de herramientas _SIG_ (**Sistemas de Información**), cabe recalcar que este es un proyecto academico sin fines de lucro, con tecnologías **Open Source**.

## Tecnologias usadas

- `Flask` - con `Python`
- `PostGis` con `PgAdmin`
